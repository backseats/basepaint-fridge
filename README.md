# The BasePaint Magnet Fridge!

A fun interactive app where you can arrange BasePaint artwork as magnets on a virtual fridge. Drag, drop, resize, and create your own compositions!

Created by [@backseats_eth](https://twitter.com/backseats_eth)

## Features

- Drag and drop BasePaint artwork onto the fridge
- Resize and reposition magnets
- Save screenshots of your creations
- Mobile and desktop support

## Development

Install dependencies:

```bash
bun install
```

Run the development server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Contributing

Want to add more magnets? Update `magnets.json` with new BasePaint artwork. Contributions are welcome!

Be excellent to everyone.
